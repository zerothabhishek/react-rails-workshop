## React Rails workshop - the React frontend

## The setup:

- Install the Yarn package manager:
  Follow the steps here:
  https://yarnpkg.com/lang/en/docs/install/

- Install the dependencies:
  `cd react/my-trello`
  `yarn install`

- Start the app:
  `yarn start`
